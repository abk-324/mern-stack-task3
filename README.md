# MERN Stack Task 3 - To-Do App

A simple MERN Stack To-Do application with full CRUD operations.

## Features

* Add Tasks
* View Tasks
* Edit Tasks
* Delete Tasks
* Light/Dark Mode
* Animated Cosmic UI

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB

## Run Backend

```bash
cd backend
npm install
node server.js
```

## Run Frontend

```bash
cd frontend
npm install
npm start
```
